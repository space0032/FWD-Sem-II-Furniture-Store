# Cozy Home Furniture Store

A simple, responsive, and functional front-end website for a fictional furniture store called "Cozy Home".

## Features

- **Responsive Design**: Adapts smoothly to different screen sizes.
- **Product Catalog**: Displays various furniture items with titles, descriptions, and prices.
- **Shopping Cart**: Users can add items to their cart, view the total price, and clear the cart or checkout.
- **About Section**: Briefly describes the store's mission and values.
- **Contact Form**: A simple form for users to send messages (with a mock success response).

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

Simply open the `index.html` file in your preferred web browser. No local server is strictly required, although it's recommended for a better development experience.
